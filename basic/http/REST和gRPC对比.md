# REST 和 gRPC对比
## Protobuf 和 JSON
Protobuf 是封装的更好的数据格式。JSON是更灵活可读性更高的数据格式。当然为了效率和封装可以使用压缩后的JSON，但是这样也就会丢失JSON的可读易用性优势。
## HTTP/2 和 HTTP 1.1
REST方式是基于HTTP的(HTTP1.1).而gRPC是使用HTTP/2协议的。
